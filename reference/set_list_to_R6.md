# set arguments in list config to r6obj

set arguments in list config to r6obj

## Usage

``` r
set_list_to_R6(config_list, r6obj_config)
```

## Arguments

- config_list:

  list containing configuration parameters

- r6obj_config:

  R6 object to set configuration in
