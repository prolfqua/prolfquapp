# add REV and zz entries - used for testing

add REV and zz entries - used for testing

## Usage

``` r
add_RevCon(stringsAll, pattern_decoys = "REV_", pattern_contaminants = "zz")
```

## Arguments

- stringsAll:

  character vector of protein IDs

- pattern_decoys:

  prefix for decoy entries

- pattern_contaminants:

  prefix for contaminant entries
